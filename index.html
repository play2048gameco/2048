How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

2048 is a sliding tile puzzle game played on a 4x4 grid. The objective is to combine numbered tiles to create a tile with the number 2048. The game starts with two tiles, each displaying a number 2 or 4. Players slide tiles in one of four directions (up, down, left, right), causing the tiles to merge if they contain the same number, doubling their value.
